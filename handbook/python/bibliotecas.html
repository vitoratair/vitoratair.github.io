The Python Standard Library


A biblioteca padrão do python é uma das coisas que fazem o python ser o python, utilizando somente bibliotecas que já vem com a linguagem é possível resolver uma série de desafios e problemas computacionais que em outras linguagem teriam que ser utilizadas bibliotecas de terceiros, a biblioteca contem módulos escritos em C que fornecem acesso a funcionalidades do sistema já contemplando códigos diferentes para os diferentes sistemas operacionais, bem como módulos escritos em python que oferecem ao programador um série de soluções padronizadas para as programações diárias. Alguns desses módulos são APIs neutras que visão abstrair e melhorar a portabilidade do código python.
Para ter acesso a biblioteca padrão do python acesso o The Python Standard Library

Além das vasta gama de módulos presentes na biblioteca padrão, com o número crescente de usuários e frameworks existe uma grande variedade de módulos python escritos pelos usuários da linguagem que você pode encontrar no github dentro de inúmeros projetos opensources que utilizam python ou no PyPI - the Python Package Index
