Tipos de dados


Tudo em python é objeto, e sua filosofia vai além de outras linguagens de programação OO, todos os valores de dados no Python são encapsulados em classes de objetos, e tudo em um programa python é um objeto acessível a partir do programa. Em C, por exemplo, para se armazenar um número inteiro deverá ter um código como o abaixo. 

int var;
var = 19;

Com o código escrito em python, simplesmente se tem 

var = 19

Por trás, o python cria um objeto integer e designa a variável para fazer referência para o novo objeto, e a mágica é, por ser uma linguagem de tipagem dinâmica, é possível atribuir valores de outros tipos na mesma variável no mesmo módulo que python irá alterar o objeto para o tipo necessário sem necessidade de interferência do programador.

Listas


Listas são como vetores em outras linguagens, são um conjunto, sequencia de valores mutáveis que podem ser acessados e indexados por um índice numérico inteiro começando de zero. Uma lista em python não necessariamente armazena somente um tipo de dado, uma lista pode armazenar dezenas de outras listas e dentro dessas podem haver dados do tipo string ou inteiros ou qualquer outro tipo de dados da linguagem.
Abaixo utilizando o interpretador python é mostrado alguns simples exemplos para manipulação de lista. 

# Criando uma lista vazia
>>> lista = []
>>> lista
[]
# Criando uma lista com 3 strings e um inteiro
>>> lista = ['element1', 'element2', 'element3', 5]
>>> lista
['element1', 'element2', 'element3', 5]
>>> lista[0:2]
['element1', 'element2']

Métodos para manipulação de listas

append

     Insere um elemento no final da lista, seu protótipo é append('elemento a ser inserido') 

>>> lista = ['element1', 'element2']
>>> lista.append('element3')
>>> lista
['element1', 'element1', 'element3']

count

     Retorna a quantidade do elemento passado por parâmetro presente na lista, seu protótipo é count('elemento a ser procurado') 

>>> lista = ['element', 'element', 'element1', 'element2']
>>> lista.count('element')
2

extend

     Insere os elementos de uma lista no final de uma segunda lista, seu protótipo é extend(lista) 

>>> lista1 = ['lista1 - Element-1', 'lista1 - Element-2', 'lista1 - Element-3']
>>> lista2 = ['lista2 - Element-1', 'lista2 - Element-2']
>>> lista1.extend(lista2)
>>> lista1
['lista1 - Element-1', 'lista1 - Element-2', 'lista1 - Element-3', 'lista2 - Element-1', 'lista2 - Element-2']

index

     Retorna o índice de um elemento dentro de uma lista, seu protótipo é index('elemento') 

>>> lista = ['lista1 - Element-1', 'lista1 - Element-2', 'lista1 - Element-3']
>>> lista.index('lista1 - Element-2')
1

insert

     Insere um elemento em uma posição desejada na lista, seu protótipo é insert(posiçao, 'elemento') 

>>> lista = ['element1', 'element2']
>>> lista.insert(1, 'ELEMENT3')
>>> lista
['element1', 'ELEMENT3', 'element2']

pop

     Deleta um elemento da lista se não especificado um índice, o último elemento é deletado, seu protótipo é pop(indice) 

>>> lista = ['element1', 'element2', 'element3', 'element4']
>>> lista.pop(1)
'element2'
>>> lista
['element1', 'element3', 'element4']
>>> lista.pop()
'element4'
>>> lista
['element1', 'element3']

remove

     Deleta um elemento da lista, é necessário indicar qual elemento deverá ser removido, seu protótipo é remove('elemento') 

>>> lista = ['element1', 'element2', 'element3', 'element4']
>>> lista.remove('element2')
>>> lista
['element1', 'element3', 'element4']

reverse

     Inverte a ordem dos elementos na lista, seu protótipo é reverse() 

>>> lista = ['element1', 'element3', 'element4']
>>> lista.reverse()
>>> lista
['element4', 'element3', 'element1']

sort

     Ordena os elementos de uma lista, seu protótipo é sort() 

>>> lista = ['element4', 'element3', 'element1']
>>> lista.sort()
>>> lista
['element1', 'element3', 'element4']

Tuplas


São também uma sequencia de valores que podem ser indexados através de um índice numérico inteiro começando em 0, as tuplas se compartam de forma muito semelhante as listas, tendo um importante diferença, são imutáveis, ou seja, uma vez criadas os dados não poderão ser alterados. 

# Criando uma tupla vazia
>>> tupla = ()
# Criando uma tupla com elementos
>>> tupla = ('element1', 'element2', 'element3', 5)
>>> tupla
('element1', 'element2', 'element3', 5)
>>> tupla[0:2]
('element1', 'element2')
>>> tupla[1:3]
('element2', 'element3')

Métodos para manipulação de tuplas

count

     Retorna a quantidade do elementos passado como parâmetro em uma tupla, seu protótipo é count('elemento') 

>>> tupla = ('element', 'element', 'element1', 'element2')
>>> tupla.count('element')
2

index

     Retorna o índice de um elemento dentro de uma tupla, seu protótipo é index('elemento') 

>>> tupla = ['element-1', 'element-2', 'element-3']
>>> tupla.index('element-2')
1

Dicinários


São sequencias que podem utilizar índices para indexação de tipos variáveis, desde que o índice seja uma variável imutável como uma string ou tupla e são implementados com o conceito de Hash Tables
Abaixo alguns simples exemplos para demostrar como é a estrutura de um dicionário. 

# Criando um dicionário vazio
>>> dic = {}
# Criando um dicionário somente com chaves
>>> dic = {}.fromkeys(['nome','endereco'])
>>> dic['nome'] = "Python"
>>> dic['endereco'] = "PSF"
>>> dic
{'endereco': 'PSF', 'nome': 'Python'}
>>> dic['nome']
'Python'

Métodos para manipulação de dicionários

# clear -> Limpa todo conteúdo do dicionário
>>> dic = {'nome': 'python'}
>>> dic
{'nome': 'python'}
>>> dic.clear()
>>> dic
{}

# copy -> Retorna um dicionário com o mesmo conteúdo.
>>> dic = {'nome': 'python'}
>>> dic1 = dic.copy()
>>> dic1
{'nome': 'python'}

# fromkeys -> Retorna um dicionário com as mesmas chaves e com o conteúdo que é passado por parâmetro.
>>> dic = {'nome': 'vitor', 'idade': 23}
>>> dic1 = dic.fromkeys(dic, 'NOVO_VALOR')
>>> dic
{'nome': 'vitor', 'idade': 23}
>>> dic1
{'idade': 'NOVO_VALOR', 'nome': 'NOVO_VALOR'}

# get -> Retorna o conteúdo da chave informado por parâmetro
>>> dic = {'nome': 'vitor', 'idade': 23}
>>> dic.get('nome')
'vitor'
