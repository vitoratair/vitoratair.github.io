O que é o Linux?

Linux não é o Linux, e sim o GNU/Linux

A maioria dos usuários GNU/Linux não sabem, ou falam que utilizam o sistema operacional Linux. Eu mesmo apesar de utilizar o Fedora a um bom tempo, sempre falei que utilizava o SO Linux e não GNU/Linux, até ter a oportunidade de assistir uma palestra do Richard Stallman no FISL 2013, a palestra era sobre a importância do software livre na sociedade e o terror dele pela Microsoft. Lá ele deixou bem claro que o copyright é importante, Linux é o kernel do sistema operacional desenvolvido por Linus Torvalds na Finlândia e não o sistema operacional completo.

O "Linux" que usamos em nossos computadores é resultado de um projeto chamado GNU, bom, tudo começa em 1985, quando o mesmo Richard Stallman cria a Free Software Foundation, uma organização sem fins lucrativos que dedica-se a eliminação das restrições sobre a cópia, redistribuição, estudo e modificação de programas de computadores. Stallman junto com FSF cria vários componentes importantes para um sistema operacional, o kernel não é um deles, então quando em 1991 Torvalds lança a primeira versão do Linux, a FSF integra o Linux no projeto GNU e nasce o que chamamos de Linux porém é o GNU/Linux, o sistema operacional completo.
