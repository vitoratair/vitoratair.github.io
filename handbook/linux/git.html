GIT

O que é?

É um sistema de controle de versão distribuido open source desenhado para velocidade e eficiência. Criado por Linux Torvalds para o desenvolvimento do kernel linux. Como características principais é possível destacar:

  • Distribuído
  • Velocidade
  • Projetado para grandes volumes de dados, branches e desenvolvedores "kenel"

Desde seu nascimento em 2005, git evoluiu e vem evoluindo constantemente, é extremamente rápido e se tornou um dos SCVs mais populares da atualidade.
