Turbinando seus scripts com AWK


AWK é uma linguagem de programação criada na década de 70 no laboratório Bell Labs tendo como objetivo deixar scripts shell com uma melhor performance e recursos. É extremamente utilizado para manipulação de arquivos e processamento de textos e comandos em sistemas Unix.
Com AWK é possível gerenciar pequenos bancos de dados pessoais, gerar relatórios, fazer experimentos e testes para mais tarde implementar se necessário em uma outra linguagem de programação compilada.

A função básica do AWK é percorrer linhas de um arquivo ou saída de um comando procurando por padrões, para cada padrão encontrado é possível tomar uma decisão e associar uma ação, sua estrutura básica pode ser definida como awk [padrão / condição] [ação]. Abaixo segue uma série de exemplos que exemplificarão o uso da linguagem.

Exemplos de uso


Para exemplos mais básicos será utilizado os próprios arquivos do SO.

Lendo arquivos

Utilização mais básica do comando, lê linha a linha do arquivo /etc/passwd 

awk '{print $0}' /etc/passwd
...
nobody:*:-2:-2:Unprivileged User:/var/empty:/usr/bin/false
root:*:0:0:System Administrator:/var/root:/bin/sh
daemon:*:1:1:System Services:/var/root:/usr/bin/false
...

Lê linha a linha do arquivo /etc/passwd, porém especifica o delimitador de campo como sendo : através da sintaxe -F : e exibi o primeiro campo, um espaço e o quinto campo do arquivo. 

awk -F : '{print $1 "  " $5 }' /etc/passwd
...
nobody  Unprivileged User
root  System Administrator
daemon  System Services
...

Lê linha a linha do arquivo /etc/passwd, especifica o delimitador de campo como sendo :, exibe o primeiro e quinto campo das linhas cujo primeiro campo inicie a partir da letra "e". 

awk -F : '$1 > "e" { print $1 " " $5}' /etc/passwd
...
nobody Unprivileged User
root System Administrator
...
