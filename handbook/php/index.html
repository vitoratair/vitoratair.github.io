O começo de tudo


PHP é uma linguagem de programação interpretada open source que foi originalmente pensada para o desenvolvimento de aplicações no lado do servidor, a fim de gerar conteúdos dinâmicos para WEB. PHP é uma linguagem extremamente simples para um iniciante, até para quem nunca programou nada em sua vida começar com PHP poderia ser uma boa opção, oferece também ótimos recursos para programadores experientes como frameworks full stack para desenvolvimento ágil e uma ótima compatibilidade com bancos de dados.

Pré-requisitos


PHP é embutido em código HTML, com isso é necessário um conhecimento básico sobre o assunto, é recomendado o site tableless iniciantes para entender os conceitos de desenvolvimento WEB com HTML. É necessário também a instalação de um servidor WEB compatível, como exemplo o apache.

Hello World


Abaixo um simples Hello World, onde será impresso no navegador somente a string Meu primeiro Hello World no mundo da WEB!!! 

<!DOCTYPE html>
<html lang="pt-br">
  <head>
    <title>Título da página</title>
    <meta charset="utf-8">
  </head>
  <body>
	<?php
	  echo "Meu primeiro Hello World no mundo da WEB!!!";
	?>
  </body>
</html>
